Thẻ hiển thị hình ảnh

Địa chỉ tuyệt đối (url - tuyệt đối):

Đỉa chỉ hình ảnh tương đối: 

            - Vào thư mục 
                + thumuc/thu-muc-con/file-hinh-anh.jpg
                + ví dụ: images/logo-2022.svg
            - ra ngoài một thư mục:
                + ../ (ra ngoài một thư mục, tính từ 
                        vị trí hiện tại)
                + ví dụ: 
                    ../Lesson01/images/logo.png

Liên kết trang

        Sử dụng thẻ liên kết:
        thẻ a 
        tag         :    a 
        attribute   :   href = url (tương đối, tuyệt đối)
                    :   title = "Nội dung văn bản"
                    :   target= 
                    
                    Value	Description
                    _blank	Opens the linked document in a new window or tab
                    _self	Opens the linked document in the same frame as it was clicked (this is default)
                    _parent	Opens the linked document in the parent frame
                    _top	Opens the linked document in the full body of the window
                    framename	Opens the linked document in the named iframe
